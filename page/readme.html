数据上传

上传三元组文件：

  1. 点击“上传三元组文件”部分中的文件选择按钮，选择要上传的文件。
  2. 点击“上传三元组文件”按钮提交表单。
  3. 等待上传完成后，在页面上将显示上传状态消息。

上传实体文件：

  1. 点击“上传实体文件”部分中的文件选择按钮，选择要上传的文件。
  2. 点击“上传实体文件”按钮提交表单。
  3. 等待上传完成后，在页面上将显示上传状态消息。
